# Security and Trademark Policy for ZEWIK™

## Reporting Misuse

If you discover:
- unauthorized use of the ZEWIK™ name,
- derivative works,
- trademark violations,
- attempts to replicate or fork the ontology,

please report it immediately.

## How to Report

Open a **private GitHub security advisory**  
or contact the repository owner (**ZewKrwi**) directly via GitHub profile.

## What to Include

- link to the violating content,
- description of the misuse,
- evidence or screenshots (if applicable).

## Response Process

Reports are reviewed by the Architect.  
Actions may include:
- takedown requests,
- DMCA filings,
- trademark enforcement,
- repository moderation.

Thank you for helping protect the integrity of ZEWIK™.
